JobNexus Application

Architecture Change: This project has been migrated to a Node.js/Express backend with EJS templates.

To run this application, you must use a Node.js environment:

  1. Open your terminal.
  2. Run npm install to install dependencies.
  3. Run npx prisma generate to setup the database client.
  4. Run npx prisma migrate dev to initialize the SQLite database.
  5. Run npm start to launch the server.
  6. Open http://localhost:3000 in your browser.
