# My Login and Register System

A full authentication system built with PHP and Node.js, including login, registration, session management, and profile features.

## Features
- User Registration & Login
- Password Hashing
- Profile Update
- Protected Pages
- Node.js + PHP Versions

## Tech Stack
- PHP (with XAMPP)
- Node.js
- MySQL Database

## How to Run (PHP)
1. Move project to XAMPP `htdocs`
2. Start Apache & MySQL
3. Visit: http://localhost/auth_app/php_app

## How to Run (Node)
1. Navigate to `node_app`
2. `npm install`
3. `npx serve public`
4. Visit http://localhost:3000
